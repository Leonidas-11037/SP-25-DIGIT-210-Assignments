You can insert video files in your html with the <video> tag. Make sure to add controls="controls" in video's attributes and put the source in the <source> tag (As an attribute, similar to img src="folder/image except it's now src="folder/video.mp4") within <video>.

As for the caption style, I used style="background:black;color:white" within <figcaption>
